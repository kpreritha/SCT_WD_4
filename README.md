# 📝 To Do List Application

A modern and responsive **To-Do Web Application** built using **HTML, CSS, and JavaScript**. This application helps users efficiently manage their daily tasks by allowing them to add, edit, delete, organize, and track tasks with date and time support.

---

## 📌 Project Overview

The To-Do Web Application is designed to improve productivity by providing a simple and intuitive interface for managing daily activities. Users can create tasks, mark them as completed, edit existing tasks, delete unwanted tasks, and organize their schedule efficiently.

This project was developed as **Task 04** during my **Web Development Internship at SkillCraft Technology**.

---

## ✨ Features

- ✅ Add New Tasks
- ✏️ Edit Existing Tasks
- 🗑️ Delete Tasks
- ✔️ Mark Tasks as Completed
- 📅 Set Due Date
- ⏰ Set Due Time
- 📱 Responsive Design
- 🎨 Modern User Interface
- ⚡ Fast and Lightweight Performance
